O que é um repositório no Git e qual a sua importância?
R:É uma área do git que contém uma pasta com os arquivos do projeto, ele se torna importante pois pode conter diversas versões do mesmo projeto, podendo acompanhar as suas mudanças feitas conforme o tempo

Qual a função do Git Bash?
R:É o meio utilizado para criar repositórios

O que faz o comando git add e qual a diferença entre git add <arquivo> e git add .?
R: Ao fazer o commit, o git add é utilizado para adicionar o arquivo no git, quando utilizado git add <arquivo> é enviado somente o arquivo em específico, já o git add . envia todos os arquivos de forma geral

Por que utilizamos o ponto (.) no comando git add .?
R:Indica que o projeto a ser enviado para o git é o que está sendo desenvolvido no momento

Explique o comando git commit e por que é importante criar uma mensagem descritiva.
R:Ele é usado parra registrar as alterações realizadas durante o projeto, é ideal que seja acompanhado com um texto, pois a descrição das ações feitas facilita o entedimento e organização

Qual a função do comando git push?
R:Ele carrega e envia os arquivos para o repositório

Qual a importância de criar commits frequentes durante o desenvolvimento de um projeto?
R:Para poder salvar cada mudança realizada, acompanhar o desenvolvimento do projeto e caso necessário realizar mudanças em certa linha do tempo do projeto

Explique a função das tags HTML e como elas estruturam uma página.
R:São utilizados para organizar e construir o projeto, também para criar uma estrutura  

O que são atributos em uma tag HTML e qual a sua função?
R:São ações e comportamentos que poderam ser realizadas de acordo com a tag

O que é o Emmet e como ele facilita o desenvolvimento de páginas HTML?
R:São abreviações que são utilizados como atalho, o que facilita e otimiza o tempo de codificação

O que é o box-sizing no CSS e por que usamos ele no reset inicial?
R:É utilizado para definir as dimenções de itens em específicos

Por que é importante utilizar tags semânticas no HTML?
R:Para poder indicar visualmente o significado de algo, o que ajuda na estruturaçãao da página, tornando fácil de identificar título, parágrafo ou outro

O que são media queries no CSS e qual a sua função?
R:É um recurso que torna seu sistema responsivo para diferentes formatos e aparelhos, tornando adaptável tanto para dispositivos móveis quanto para computadores

Qual a função do GitHub Pages?
R:É uma função do github que hospeda, torna acessível e visualizável o projeto realizado.

O que é o Flexbox no CSS e para que ele é utilizado?
R:É um método que é utilizado para a criação e planejamento de diferentes layout, o que facilita o processo de design

Qual a diferença entre justify-content e align-items em um container Flexbox?
R:Podem ser utilizado para alinhar o item na faixa horizontal e o outro na vertical, e também define como o item será alinhado

Como o flex-grow funciona e qual o seu propósito no layout Flexbox?
R:Quando á um elemento flex, ele regula a propriedade de crescimento deste elemento flex

Explique a diferença entre flex-direction: column; e flex-direction: row;.
R: "flex-direction: column" alinha seus itens em uma coluna, um abaixo do outro, já o "row" alinha em uma liinha que posiciona cada elemento um na frente do outro.
